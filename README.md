PharmaLens is a **menu-driven Pharma Management System** developed using **Python**.

## Features

- Add new medicines
- View existing medicines
- Update medicine details
- Remove medicines
- Visualize stock levels (Bar Graph)
- Visualize monthly sales trends (Line Graph)
- Persistent data storage using CSV files
- Clean modular package structure
